## Instalação

```bash
# Clone o repositório
$ git clone https://github.com/manoguii/dt-money.git
# Entre no repositório
$ cd dt-money
# Criar as variáveis ambiente em
$ npm install
# Execute a aplicação
$ npm run start
```

## Sobre

O dt-money é um projeto para gerenciar despesas e receitas. 🚀
