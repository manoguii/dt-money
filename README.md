## Instalação

- Clone o repositório
```bash
$ git clone https://github.com/manoguii/dt-money.git
```

- Instale as dependências
```bash
$ npm install
```

- Execute o banco de dados
```bash
$ npm run dev:server
```

- Execute a aplicação
```bash
$ npm run dev
```

## Sobre

O dt-money é um projeto para gerenciar despesas e receitas. 🚀
